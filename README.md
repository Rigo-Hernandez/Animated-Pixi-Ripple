# Animated-Pixi-Ripple
An animated ocean ripple effect using PixiJS 


<img src="./Demo.gif">

## Usage

```PixiJS
        function animate() {
            displacementSprite.x += 10;
            displacementSprite.y += 4;
            requestAnimationFrame(animate);
        }

```
